## Why to learn React?
1. Hype, job, trend, build UI
2. makes easy to manage and build complex front end
## When should I learn React?
1. After mastering JS
2. Most projects do not need react in initial phase
## Why react was created?
1. Facebook ghost/ phantom message problem: top chat icon would have total messages unread, after chat, icon used to stay there
2. No consistency in UI
3. State managed by JS and UI managed by DOM (Document Object Model)
4. Khan Academy and Unsplash started to develop in React
## React Learning Process
1. In this course, we will make projects
2. Todo App, Calculator, GitHub API 
## Topics to Learn
1. Core of react (state or UI manipulation, JSX(HTML written in JavaScript))
2. Component reusability
3. Reusing of component (Props)
4. How to propagate change (Hooks -> useState, useEffect, useCallback)
## Additional Topics to Learn in React
1. Router (via react-router library)
2. State Management (Redux, Redux Toolkit, Zustand, context API)
3. Class based component rarely used in modern systems, present in legacy systems
4. BAAS Apps (Social Media Clone, E-commerce App)
5. freeapi.app (open source to learn react)
## After React
1. React is not a complete solution in most cases as there is no SEO
2. Framework: NextJS, Gatsby, Remix
